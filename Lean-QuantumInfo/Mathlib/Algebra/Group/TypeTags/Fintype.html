    Finite types with addition/multiplications #

    This file contains basic results and instances for finite types that have an addition/multiplication operator.

    Main results #

    • Fintype.decidableEqMulEquivFintype: MulEquivs on finite types have decidable equality

    • Multiplicative.fintype: a finite type under addition is also finite under multiplication
