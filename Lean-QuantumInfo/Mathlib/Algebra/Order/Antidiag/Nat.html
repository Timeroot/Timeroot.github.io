    Sets of tuples with a fixed product #

    This file defines the finite set of d-tuples of natural numbers with a fixed product n as Nat.finMulAntidiagonal.

    Main Results #

    • There are d^(ω n) ways to write n as a product of d natural numbers, when n is squarefree (card_finMulAntidiagonal_of_squarefree)

    def Nat.finMulAntidiag (d n : ℕ) :
    Finset (Fin d → ℕ)

    The Finset of all d-tuples of natural numbers whose product is n. Defined to be ∅ when n = 0.
