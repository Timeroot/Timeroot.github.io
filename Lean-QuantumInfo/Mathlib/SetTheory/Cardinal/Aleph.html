    Omega, aleph, and beth functions #

    • The function Ordinal.preOmega enumerates the initial ordinals, i.e. the smallest ordinals with any given cardinality. Thus preOmega n = n, preOmega œâ = œâ, preOmega (œâ + 1) = œâ‚ÇÅ, etc. Ordinal.omega is the more standard function which skips over finite ordinals.
    • The function Cardinal.preAleph is an order isomorphism between ordinals and cardinals. Thus preAleph n = n, preAleph œâ = ‚Ñµ‚ÇÄ, preAleph (œâ + 1) = ‚Ñµ‚ÇÅ, etc. Cardinal.aleph is the more standard function which skips over finite ordinals.

    The following notations are scoped to the Ordinal namespace.

    The following notations are scoped to the Cardinal namespace.

    • ‚Ñ∂_ o is notation for beth o. The value ‚Ñ∂_ 1 equals the continuum ùî†, which is defined in Mathlib.SetTheory.Cardinal.Continuum.
