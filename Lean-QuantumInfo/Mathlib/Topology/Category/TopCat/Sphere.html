    Euclidean spheres #

    This files defines the n-sphere 𝕊 n and the n-disk 𝔻 as objects in TopCat. The parameter n is in ℤ so as to facilitate the definition of CW-complexes (see the file Topology.CWComplex).

    source
    noncomputable def TopCat.sphere (n : ℤ) :
    TopCat

    The n-sphere is the set of points in ℝⁿ⁺¹ whose norm equals 1, endowed with the subspace topology.

    Equations
    • TopCat.sphere n = TopCat.of (ULift.{?u.16, 0} ↑(Metric.sphere 0 1))
